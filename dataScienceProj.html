Sample size tutorial for AB test

During the setting phase of an AB test using frequentistic approach, one important step is to understand how long the test should run to be able to detect eventual specific effect of change in the test variant or to be able to conclude that the change did not make any differences. In general we rely on online calculators for the sample size but different calculator can give very different results. In this tutorial I collect what I found in literature and I compile all the basis behind the sample size formula and how it can be implemented in python. Find here the tutorial
